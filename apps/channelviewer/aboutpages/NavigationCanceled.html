Navigation Canceled
+ More information

ChannelViewer was unable to connect to the Web page you requested because navigation was canceled. There are several reasons why this might happen:

  • You clicked the Stop button before the page finished loading.

  • The page might not be available at this time.

+ Helpful Hints

  • To try again to open the Web page, click the Back button, and then click the link again.

  • If the Web page you requested is still taking too long to display, try clicking the Stop button, and then clicking Refresh.
