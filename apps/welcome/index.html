Setup System Plugin

For better usability, installing the system plugin is highly recommended.

It allows better integration between your system and the wallpaper, such as opening a new browser window from the wallpaper, receiving your keyboard input directly, or controlling the currently playing media.

Click for more information, including the setup instructions.
