Spotify integration
  • Enable extended Spotify integration if you think there are too many missing lyrics, and if your Spotify client uses a non-English language.
  • ModernActiveDesktop will fetch the English name of the song and artist from Spotify, and use it to search for lyrics. This will increase the chance of finding the correct lyrics.
  • You need to sign in to Spotify to enable this feature.
  • To sign in, you need to have the system plugin installed. See this guide to install the system plugin.
    •
  • Currently hidden as the Spotify client ID has not yet been approved for the extended quota mode by Spotify, unless the debug mode is enabled.
  • If you wish to use this feature right now, exit the ModernActiveDesktop System Plugin, and run systemplugin\MADSysPlug.exe with the --spotify <client_id> argument.
  • Go to the dashboard in developer.spotify.com, create a new app, put http://localhost:3031/spotify/callback in the Redirect URIs, and get the client ID there.
