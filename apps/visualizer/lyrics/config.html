Spotify integration
  • Enable extended Spotify integration if you think there are too many missing lyrics, and if your Spotify client uses a non-English language.
  • ModernActiveDesktop will fetch the English name of the song and artist from Spotify, and use it to search for lyrics. This will increase the chance of finding the correct lyrics.
  • You need to sign in to Spotify to enable this feature.
  • To sign in, you need to have the system plugin installed. See this guide to install the system plugin.

Miscellaneous
  • Enable this option if your music player does not provide timeline information, or if your player provides unreliable timeline information (e.g. browsers).
  • Enable Visualizer -> View -> Estimate Timeline if your player provides reliable timeline info but doesn't update it every seconds (e.g. Spotify).
