    CORS proxy

    You can change which proxy to use for fetching resources.

    Proxy:

    Advanced
    • ChannelViewer will try to load all pages with a CORS proxy.
    • Advanced features will be enabled for all Web pages, including those from another origin.
    • Web pages that doesn't allow embedding normally will also be force loaded.
    • Do note that some pages might not function correctly, especially those with complex scripts.
