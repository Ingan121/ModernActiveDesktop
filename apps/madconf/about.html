    Credits

    ModernActiveDesktop uses the following projects:

    * 98.css - A CSS library for building Windows 98-like interfaces
    * XP.css - Used for the Windows XP style
    * 7.css - Used for the Windows Aero style
    * JS Paint - A web-based remake of MS Paint
    * js-solitaire - A web-based remake of MS Solitaire
    * X-Frame-Bypass - Allows ChannelViewer to load more webpages
    * marked - A JavaScript library for parsing markdown docs
    * Electron - Used for the system plugin
    * minimist - A Node.js library for parsing command line arguments
    * WindowsMediaController - Used by MediaControlCLI for media controls
    * And many assets from Microsoft Windows
    * And various codes snippets from Stack Overflow
